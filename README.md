## 👻 Snapchat Guide

Key privacy settings:
- **Ghost Mode**: Hide location from all friends
- **Contact Me**: Only friends (not everyone)
- **See Me in Quick Add**: Off (prevents random adds)
- **Two-Factor**: SMS + Authenticator app
